# Projeto IoT com ESP32

Repositório dedicado ao estudo da disciplina de **Internet das Coisas (IoT)** no **IFPB**, utilizando o microcontrolador **ESP32** e o framework **ESP-IDF**.

---

## Objetivo

Desenvolver aplicações embarcadas conectadas à rede, explorando conceitos fundamentais de IoT, como:

- Conexão Wi-Fi (modo Station)
- Comunicação em rede
- Sincronização de data e hora via SNTP
- Integração com serviços externos

---

## Tecnologias Utilizadas

- ESP32
- ESP-IDF
- Linguagem C
- Protocolo Wi-Fi
- SNTP (Simple Network Time Protocol)
